# Getting Started

## Run 

Run **TaaSpringPart3Application**

## H2 Console

Go to [localhost:8080/h2](localhost:8080/h2) to access the h2 console.

#### Settings :
|Settings|Value|
|---|---|
|Saved Settings|Generic H2|
|Saved Name|Generic H2|
|Driver Class|org.h2.Driver|
|JDBC URL|jdbc:h2:mem:tpspring|
|Username|sa|
|Password|   |
